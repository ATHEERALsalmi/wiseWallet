# 🏦 wiseWallet - تطبيق مالي ذكي للمستخدم السعودي

wiseWallet هو تطبيق مالي مبتكر مصمم خصيصاً لتلبية احتياجات المستخدم في المملكة العربية السعودية، حيث يساعد على إدارة المصاريف اليومية والمدخرات بذكاء وسهولة باستخدام تقنيات برمجية حديثة.

---

## 🚀 المميزات الرئيسية (Key Features)
* إدارة الميزانية الذكية: تتبع المصاريف والدخل بشكل دوري وبواجهات تفاعلية سهلة.
* تحليلات وتقارير: توليد رسوم بيانية وتقارير مخصصة توضح سلوك الإنفاق وتساعد على اتخاذ قرارات مالية أفضل.
* الاستشارات المالية (AI Advisor): نظام مستشار ذكي يقدم نصائح مخصصة لتحسين الادخار وتقليل الهدر المالي بناءً على السلوك المالي للمستخدم.
* أمان وحماية عالية: حماية بيانات المستخدم المالية وتشفيرها لضمان الخصوصية التامة.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

### 💻 الواجهة الأمامية (Frontend)
* HTML5 & CSS3: لبناء وتصميم واجهات المستخدم لتكون متجاوبة (Responsive) وسلسة.
* JavaScript (Vanilla): لإضافة التفاعلية والتحكم في عناصر الواجهة بشكل ديناميكي.

### ⚙️ الواجهة الخلفية (Backend)
* Node.js & Express.js: لبناء سيرفر سريع وخفيف للتحكم بالعمليات والبيانات.
* مكونات النظام الخلفي الأساسية:
  * auth.js: نظام توثيق وتسجيل دخول آمن للمستخدمين.
  * advisor.js: نظام الاستشارات الذكي لتقديم التوصيات المالية المخصصة.
  * classifier.js: مصنف مالي ذكي لتصنيف المصاريف تلقائياً.
  * healthScore.js: نظام احتساب اللياقة والتقييم المالي العام للمستخدم.
