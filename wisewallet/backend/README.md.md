# wiseWallet — MVP كامل (Frontend + Backend)

تطبيق مالي ذكي للمستخدم السعودي، حسب خطة التنفيذ المرفقة (هاكاثون Tuwaiq × Alinma).

## التشغيل السريع

**1. شغّل الـ Backend:**
```bash
cd backend
node server.js
```
يعمل على `http://localhost:3001` — لا يحتاج `npm install` (بدون أي مكتبات خارجية).

**2. افتح الـ Frontend:**
افتح ملف `frontend/index.html` مباشرة في المتصفح (أو عبر أي خادم ملفات ثابتة بسيط مثل `npx serve frontend`).

> يجب تشغيل الـ Backend أولًا قبل استخدام الواجهة، لأن كل الشاشات (تسجيل الدخول، العمليات، المؤشر المالي، المستشار) تعتمد على استدعاءات API حقيقية.

## المزايا المطبقة (مطابقة لنطاق الـ MVP في خطة التنفيذ)

- ✅ تسجيل / دخول مستخدم (كلمات مرور مُجزّأة بـ scrypt)
- ✅ إدخال دخل ومصاريف يدويًا
- ✅ تصنيف المصاريف تلقائيًا بالكلمات المفتاحية (مطاعم، مواصلات، فواتير، تسوق...)
- ✅ مؤشر الصحة المالية (Financial Health Score) بناءً على نسبة الادخار والتحكم بالإنفاق
- ✅ لوحة تحكم: إجمالي الدخل/المصاريف، نسبة الادخار، توزيع المصاريف حسب الفئة
- ✅ مستشار مالي ذكي (توصيات مبنية على قواعد منطقية، قابلة للاستبدال بنموذج LLM لاحقًا دون تغيير الواجهة)
